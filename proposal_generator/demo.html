Troubleshooting

  • 404 Error: Ensure Apache is running in XAMPP (not Tomcat)
  • PHP Errors: Check PHP 8.2+ is enabled in XAMPP
  • Blank Page: Check error logs in C:\xampp\apache\logs\error.log
  • CSS Missing: Ensure styles.css exists in the same folder
